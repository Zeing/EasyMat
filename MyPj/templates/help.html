EasyMat使用帮助

EasyMat为一款类Matlab的绘图应用，主程序采用类命令行的窗口，可以生成png、pdf等多种格式的函数图像，可以满足绝大部分的函数绘图需求，从而为用户提供了极大的便利。

用户可以在左侧的输入区域逐行输入绘图命令并回车，然后在右侧输入想要得到的文件名并选择文件类型，然后输入回车，即可生成函数图像。用户可以在右侧可以选择需要输出的文件类型，默认为png格式。下面是主页面命令行的输入示例：

>>> Welcome to EasyMat!
>>> hello world
>>> hello world
>>> hello world
>>> 

然后输入draw命令即可进行绘图。若输入clear命令，则清除之前的所有命令，重新开始输入。可生成如下函数图像：

>>> t1 = arange(0.0, 0.5, 0.1)
>>> t2 = arange(0.0, 0.5, 0.02)
>>> figure(1)
>>> subplot(211)
>>> plot(t1, exp(-t1) * cos(2 * pi * t1, 'bo', t2, exp(-t2) * cos(2 * pi * t2), 'k'))
>>> subplot(212)
>>> plot(t2, cos(2 * pi * t2), 'r--')

>>> mu, sigma = 100, 15
>>> x = mu + sigma * random.randn(10000)
>>> # the histogram of the data
>>> n, bins, patches = hist(x, 50, normed=1, facecolor='g', alpha=0.75)
>>> xlabel('Smarts')
>>> ylabel('Probability')
>>> title('Histogram of IQ')
>>> text(60, .025, r'$\mu=100,\ \sigma=15$')
>>> axis([40, 160, 0, 0.03])
>>> grid(True)

>>> fig = figure
>>> ax = Axes3D(fig)
>>> X = arange(-4, 4, 0.25)
>>> Y = arange(-4, 4, 0.25)
>>> X, Y = meshgrid(X, Y)
>>> R = sqrt(X**2 + Y**2)
>>> sin(R)
>>> ax.plot_surface(X, Y, Z, retride = 1, cstride = 1, cmap = 'hot')

EasyMat语法与MATLAB相似但for循环需按下面示例方式书写：

>>> for 条件:
>>>    执行语句
注意：执行语句前需要四格空格或者一个缩进。
